# ❌⭕ Tic-Tac-Toe Game – Level 3 Task

## 📄 Description
A **two-player console-based game** where players take turns marking X or O on a 3x3 grid until:
- One wins
- Or the game is a draw

## Concepts Used
- 2D arrays
- Game loop and input validation
- Win condition checking
- Turn switching logic

## How to Run
```bash
g++ main.cpp -o tictactoe
./tictactoe
